# The Prototype Bet

**What I Built**
I built a clickable web app prototype for Waymo Mobility Copilot — an AI-powered ride planning assistant that helps riders plan safer, smarter, and more reliable autonomous rides.
The prototype demonstrates how a user can enter a trip request, receive an AI-generated ride recommendation, compare route options, understand pickup safety, and book a Waymo ride with more confidence.

**Tool Used**
ChatGPT / HTML clickable prototype

**Prototype Link**
https://chat.openai.com/ 

**AI Value Archetype**
Copilot / Planner

**The Bet in One Sentence**
Waymo can increase rider trust and booking confidence by using AI to recommend the safest pickup point, explain route confidence, compare ride options, and guide passengers through the autonomous ride experience before they book.

**Kill Criteria**
I would stop this bet if users do not find the AI recommendations helpful, if the explanations do not increase trust in autonomous rides, or if users still prefer booking a traditional rideshare without reviewing Waymo’s pickup safety, route confidence, and arrival reliability guidance.
